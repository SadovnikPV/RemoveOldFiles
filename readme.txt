REMOVE OLD FILES
v 0.1

Программа удаляет устаревшие папки и файлы.
 - В данный момент программа работает только в директории "old", которая должна находиться на одном уровне с папкой самой программы.
 - В данный момент программа удаляет файлы старше 10 дней.
 - Количество дней считается не с даты создания файла, а с даты его последнего изменения!
 - Программа ведет лог основных операций в файле log.txt, находящийся в корневой директории. Ознакомтесь с ним при выявлении неполадок.

Комментарии для разработчика:
Чтобы программа считала количество дней с даты создания файла, а не с даты последнего изменения, надо в последней функции (get_file_or_folder_age) изменить выражение 
os.stat(path).st_mtime 
на 
os.stat(path).st_сtime.
